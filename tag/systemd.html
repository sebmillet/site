Quelques commandes systemd
sam. 07 novembre 2020

(D'après Linux Reference numéro 7, juin - juillet - août 2020)

  • Détails sur le démarrage (durées de chargement) :
systemd-analyze
  • Lister les services lancés au démarrage :
systemctl list-unit-files --state=enabled
  • Pour afficher les services qui ont pris le plus de temps à démarrer :
systemd-analyze blame
  • Pour lister les timers (exécutions périodiques) :
systemctl list-timers --all
