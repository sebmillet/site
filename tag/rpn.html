pRPN pour "Portable RPN"

Calculatrice fonctionnant en notation polonaise inverse, reprenant les fonctionnalités et le langage de programmation des calculatrices HP. Code C++ / wxWidgets 3.0, disponible sur Windows et Linux.

La version actuelle est la 0.6.1, du 3 novembre 2020.
