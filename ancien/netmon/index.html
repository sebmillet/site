netmon
netmon effectue des tests de connexion TCP à intervalles réguliers et écrit le résultat dans un fichier HTML.
netmon fournit un serveur web minimaliste pour afficher le statut.
netmon peut également effectuer des "email loop" => envoi d'email et réception depuis la boîte aux lettres cible, pour vérifier que toute la chaîne entre l'envoi et la réception fonctionne.

Windows
Le fichier netmon-1.1.5.zip contient l'exécutable netmon.exe, prêt à l'emploi.

Linux
Télécharger le fichier netmon-1.1.5.tar.gz, extraire son contenu et exécuter
./configure
make
sudo make install
