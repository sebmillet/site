tcprelay
Retransmet le trafic TCP entre un serveur et un client et logue les données échangées.
Fonctionne sur Windows et Linux.

Windows
L'exécutable tcprelay.exe est prêt à l'emploi.

Linux
Télécharger le fichier tcprelay-1.3.2.tar.gz, extraire son contenu, aller dans le répertoire extrait et exécuter la séquence ./configure, make et make install. 

tar -zxvf tcprelay-1.3.2.tar.gz
cd tcprelay-1.3.2/
./configure
make
sudo make install
