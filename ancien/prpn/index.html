pRPN pour "Portable RPN".
Calculatrice fonctionnant en notation polonaise inverse, reprenant les fonctionnalités et le langage de programmation des calculatrices HP. Code C++ / wxWidgets 2.8, disponible sur Windows et Linux.
La version actuelle est la 0.5.5, du 7 mai 2012.
