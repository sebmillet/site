smtprec

dim. 19 juillet 2015

smtprec est un serveur SMTP qui logue la conversation SMTP et enregistre les messages reçus. Il ne relaie pas les messages, il doit donc être utilisé pour faire du débugage. Fonctionne sur Windows et Linux.

Windows

L'exécutable smtprec.exe est prêt à l'emploi.

Linux

Télécharger le fichier smtprec-1.2.tar.gz, extraire son contenu et exécuter

./configure
make
make install
